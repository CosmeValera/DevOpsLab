1. To create the release execute one of these:
```
helm install devopslab ./deployments/helm/devopslab -f ./deployments/helm/devopslab/values-dev.yaml
helm install devopslab ./deployments/helm/devopslab -f ./deployments/helm/devopslab/values-prod.yaml
```

Once the release is cretead, you have 2 options to access the webapp:
1) Use 'minikube addons enable ingress' and 'minikube tunnel'
  1.1) You should be able to access the front app in 'localhost' and the back in 'localhost/api'
2) Use 'port-forward'
  2.1) You should be able to access the front app in 'localhost:3000' and the back in 'localhost:3001'
